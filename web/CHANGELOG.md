# CRML Studio Changelog

This changelog covers CRML Studio (the web UI) in `web/`.

CRML Studio is versioned independently from:

- `crml-lang` (language/spec)
- `crml-engine` (reference engine / CLI)

## 0.1.0

### Added
- Next.js web UI for validating and exploring CRML models
- Validator view backed by the Python API route (uses `crml-engine` server-side)
- Examples gallery for loading models from `examples/`
- Simulation UI for running the reference engine and displaying results
