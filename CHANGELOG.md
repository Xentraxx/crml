# Changelog

This repository contains separately versioned components:

- `crml-lang` (language/spec): models + schema + validation + YAML IO
- `crml-engine` (reference engine): CLI + runtime/simulation
- `web/` (CRML Studio): Next.js interface for validation/simulation

Use the package-specific changelogs:

- `crml_engine/CHANGELOG.md`
